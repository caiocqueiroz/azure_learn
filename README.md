# Azure Learn

Repositório de estudos para entrevista técnica/executiva com foco em **Azure Platform Engineering**.

## Objetivo

Usar este repositório como um guia prático para estudar:

- Azure Container Apps e estratégia de containers;
- networking enterprise no Azure;
- migração e modernização de bancos de dados;
- Terraform e platform engineering;
- CI/CD e GitOps;
- FinOps;
- segurança, identidade e governança;
- comunicação executiva para entrevistas.

## Guia de estudos

Comece pelo índice principal:

- [Plano de Estudos — Azure Platform Engineering](estudos/azure-platform-engineering/README.md)

## Como usar este repositório

1. Siga os blocos na ordem sugerida.
2. Leia os conceitos principais de cada tema.
3. Pratique as respostas modelo em voz alta.
4. Monte histórias reais usando a estrutura `Contexto → Problema → Opções → Decisão → Trade-offs → Resultado`.
5. Revise as perguntas prováveis antes da entrevista.
