# Bloco 8 — Executive Communication

**Tempo sugerido:** 45 min

## Objetivo

Preparar histórias reais que demonstrem:

- ownership;
- arquitetura;
- pragmatismo;
- segurança;
- comunicação;
- impacto no negócio.

## Estrutura para contar histórias

Use:

```text
Contexto
Problema
Opções avaliadas
Decisão
Trade-offs
Resultado
```

Ou:

```text
Situation
Decision
Trade-off
Outcome
```

## Resposta modelo

> “In a previous environment, we had a challenge with X. The main risk was Y. We evaluated options A and B. We chose B because it reduced operational overhead and improved governance, although it required some upfront standardization. The outcome was better reliability, faster delivery and lower operational risk.”

## Histórias reais para preparar

Prepare pelo menos 5 histórias:

1. Migração on-prem para Azure;
2. ARC / runners / Kubernetes;
3. FinOps dashboard;
4. Networking / enterprise governance;
5. CI/CD modernization;
6. Incidente ou problema crítico em produção;
7. Decisão arquitetural com trade-offs claros.

Para cada história, tenha:

- contexto;
- desafio;
- sua decisão;
- trade-offs;
- resultado mensurável ou qualitativo.
